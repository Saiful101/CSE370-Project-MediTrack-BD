# MediTrack BD

This project implements a relational database system for a banking scenario.  
It contains customer, account, branch, loan, depositor, and borrower data.

## Project Idea
The system tracks customers, their accounts, and loans across branches.

## Features
- Store customer details
- Manage bank branches
- Track accounts and balances
- Track loans and borrowers
- SQL queries using JOINs, GROUP BY, aggregation, and subqueries
